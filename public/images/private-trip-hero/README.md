# 우리여행 히어로 전용 이미지

이 **폴더 루트에만** 이미지 파일을 두면 `/travel/overseas/private-trip` 상단 히어로가 **파일명 순**으로 자동 슬라이드합니다.

- 지원 확장자: `.jpg` `.jpeg` `.png` `.webp` `.gif` `.avif`
- 하위 폴더는 스캔하지 않습니다.
- 여기에 **한 장이라도** 있으면 `public/data/private-trip-hero-slides.json` 설정보다 **이 폴더가 우선**입니다.

## 관리자 업로드 (권장)

**메인 허브 이미지** 페이지 → 「우리여행 히어로」에서 파일을 올리면 서버가 자동으로:

- **비율**: 와이드 히어로용 **1920×640**에 맞춰 중앙 기준 **cover** 크롭  
- **형식·용량**: **WebP**로 통일 (품질 약 80)

직접 이 폴더에 WebP를 넣어도 됩니다. SFTP로만 넣을 때는 위 비율에 맞춰 두면 화면이 더 안정적입니다.
